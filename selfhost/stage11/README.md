# Stage 11: Full C Compiler

## What Will Be Here

Expansion of the Stage 08 subset-C compiler toward full C language
support. Features like `float`/`double`, full preprocessor, `switch`,
`goto`, wider type system, etc.

## What It Will Need

- Stage 08 cc-min as starting point
- Language and runtime feature roadmap

## What It Will Produce

- A C compiler capable of compiling real-world C programs

## Status

Not started. Depends on Stage 08 completion.
